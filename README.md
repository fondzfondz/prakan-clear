# ประกันเคลียร์ Demo

ตัวอย่างเว็บ MVP สำหรับวิเคราะห์แบบประกันชีวิต

## มีอะไรใน Demo นี้

- หน้าอัปโหลดใบเสนอขาย
- โหมดวิเคราะห์ 1 แบบ
- โหมดเปรียบเทียบ 2 แบบ
- Rules Engine ที่ใช้คุม AI
- เครื่องคำนวณ IRR เบื้องต้น
- หน้าผลวิเคราะห์ตัวอย่าง
- ฟอร์มเก็บเบอร์ Lead ตัวอย่าง
- โครง Backend API สำหรับต่อ GPT ภายหลัง

## วิธีรัน

```bash
npm install
npm run dev
```

## สิ่งที่ยังไม่ได้ต่อจริง

- GPT API
- อ่าน PDF จริง
- เก็บเบอร์ลง Database
- ระบบ Login
- ระบบ Admin

## หลักการสำคัญ

ห้ามใส่ API key ใน `src/App.jsx`
ให้เรียกผ่าน `/api/analyze-insurance.js` เท่านั้น
